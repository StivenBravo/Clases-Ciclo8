# Sistema de Librería - Node.js

Sistema web para gestión de una librería con catálogo de libros, comics y revistas.

## Características

- ✅ Catálogo de libros con título, autor y precio
- ✅ Catálogo de comics y revistas con ediciones
- ✅ Carrito de compras con cálculo de IGV (18%)
- ✅ Agregar nuevos libros al catálogo
- ✅ Historial de ventas registradas
- ✅ Arquitectura de microservicios con Node.js

## Cómo ejecutar

1. Abrir 3 terminales en la carpeta del proyecto

2. En la terminal 1:
   node server1.js

3. En la terminal 2:
   node server2.js

4. En la terminal 3:
   node server3.js

5. Abrir `index.html` en el navegador

## Tecnologías

- Node.js (módulo http nativo)
- Bootstrap 5
- JavaScript ES6+
- JSON para persistencia de datos
